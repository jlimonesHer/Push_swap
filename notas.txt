$ sa : swap a - intercambia los dos primeros elementos encima del node a. No hace
nada si hay uno o menos elementos.


$ sb : swap b - intercambia los dos primeros elementos encima del node b. No hace
nada si hay uno o menos elementos.


$ ss : swap a y swap b a la vez.


$ pa : push a - toma el primer elemento del node b y lo pone encima del node a.
No hace nada si b está vacío.


$ pb : push b - toma el primer elemento del node a y lo pone encima del node b.
No hace nada si a está vacío.


ra : rotate a - desplaza hacia arriba todos los elementos del node a una posición,
de forma que el primer elemento se convierte en el último.


rb : rotate b - desplaza hacia arriba todos los elementos del node b una posición,
de forma que el primer elemento se convierte en el último.


rr : rotate a y rotate b - desplaza al mismo tiempo todos los elementos del node
a y del node b una posición hacia arriba, de forma que el primer elemento se
convierte en el último.


rra : reverse rotate a - desplaza hacia abajo todos los elementos del node a una
posición, de forma que el último elemento se convierte en el primero.


rrb : reverse rotate b - desplaza hacia abajo todos los elementos del node b una
posición, de forma que el último elemento se convierte en el primero.


rrr : reverse rotate a y reverse rotate b - desplaza al mismo tiempo todos
los elementos del node a y del node b una posición hacia abajo, de forma que
el último elemento se convierte en el primero.
